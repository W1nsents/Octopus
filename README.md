# Octopus
Advanced AutoClicker For Minecraft Made In C#.
- Possible Problems:
  - If there is a package error (Costura installation)
  - If there is a form error (Installing libraries: Siticone UI, Guna UI2)
# Features
- Left Clicker
  - Average CPS
  - Blatant Mode
  - Hotkey
  - Only in-game
  - Shift Disable
  - RMB Lock
  - Jitter
    - Jitter Range
  - Blockhit
     - Blockhit CPS
- Right Clicker
  - Blatant Mode
  - Only in-game
  - Shift Disable
  - Hotkey
  - Eat / Bow
- Settings
  - Hide Autoclicker
